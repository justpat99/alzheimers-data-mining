# Alzheimer's Disease Data Mining & Prediction

## Overview
This project analyzes patient data to identify risk factors and predict Alzheimer's disease using data mining and machine learning techniques.

## Project Objectives
- Identify key factors influencing Alzheimer’s disease
- Perform classification and clustering
- Analyze patient risk patterns

## Tools & Technologies
- Orange (Data Mining)
- Jupyter Notebook
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)

## Methodology
### Phase 1: Orange
- Data cleaning & preprocessing
- Feature engineering
- Model training & evaluation

### Phase 2: Python
- Missing value handling
- PCA & t-SNE for dimensionality reduction
- Data visualization and pattern analysis

## Results
- Decision Tree achieved higher performance than kNN (Accuracy ~0.86)
- Key influencing factors include MMSE score and behavioral indicators
- Clear clustering patterns observed using t-SNE visualization
